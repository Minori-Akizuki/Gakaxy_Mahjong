# galaxy-mahjong

[銀河麻雀](https://youtu.be/IDaKM7eU7zE) という麻雀ルールの待ち牌解析ができるページです。
麻雀ルール部分は後々 NPM に切り出す予定ですが、現在は全て `lib` ディレクトリに入っています。
麻雀ゲームとか作りたい人はフォークしてそこだけ切り出すのもアリです。
